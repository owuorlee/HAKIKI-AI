# 🛡️ HAKIKI AI | Sovereign Payroll Defense

> **"Trust is no longer manual. It is sovereign."**

## 🚨 The Problem
The "Ghost Worker" crisis is not an HR error; it is a crime scene. Kenya loses **billions annually** to payroll syndicates that hide in the noise of 50,000+ employee records. Manual audits are too slow to catch them.

## ⚡ The Solution
HAKIKI AI is an autonomous forensic engine. It replaces random sampling with **Graph Neural Networks (GNNs)** to mathematically isolate fraud rings.

## 🧩 Key Capabilities

### 1. The "Red Mode" (Graph Interdiction)
Unlike standard dashboards, HAKIKI visualizes payroll as a **Network Graph**.
- **Nodes:** Employees, Bank Accounts, Devices.
- **Edges:** Financial transactions.
- **The Breakthrough:** When "Suspect Mode" is toggled, the system filters 99% of noise, revealing only **"Ghost Families"**—clusters of distinct IDs funneling cash to a single destination.

### 2. Sovereign Assistant (Context-Aware AI)
A deterministic forensic chatbot that explains *why* a suspect was flagged.
- *"Why is Muthama flagged?"* -> *"Salary variance > 4.2 sigma + Shared Bank Account."*

### 3. The Sentinel (Biometric Prevention)
A mobile-first "Active Liveness" gatekeeper.
- **Geofenced:** Must be at the duty station (JKUAT Juja).
- **Biometric:** Requires active gestures (Smile, Turn Left) to prevent photo spoofing.
